# Architecture Decision Records

Short, immutable records of significant technical decisions — the *why* behind
the codebase. Each ADR has **Context**, **Decision**, **Consequences**, and
**Alternatives Considered**. New decisions get a new numbered file; superseded
ADRs are marked, not deleted.
